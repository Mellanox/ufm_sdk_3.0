To list plugin pods for this release:

  kubectl get pods -n {{ include "ufm-plugin.namespace" . }} -l app.kubernetes.io/instance={{ .Release.Name }}

Or by plugin label:

  kubectl get pods -n {{ include "ufm-plugin.namespace" . }} -l plugin-name=<plugin-name>

To check logs for a plugin:

  kubectl logs -n {{ include "ufm-plugin.namespace" . }} -l app={{ .Values.ufmFullname }}-plugin-<plugin-name> -f

Replace <plugin-name> with the plugin name (e.g. log-streamer). Use the same value as in your plugins list (underscores in names are rendered as hyphens in pod names).
